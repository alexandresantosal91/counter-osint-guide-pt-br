Excluindo suas próprias contas

Apesar de a Internet lembrar de tudo, a exclusão de contas permanece como um dos meios mais fortes contra a inteligência básica sobre você. Parece que este é um tópico trivial, mas existem várias nuances importantes:

Antes de excluir:

  1. Solicite ao serviço para fazer upload dos seus dados pessoais pelo GDPR. Isso ajudará nas próximas etapas e dará a você uma compreensão do que o serviço sabe sobre você.
  2. Renomeie sua conta, exclua/mude fotos, espere alguns dias. Isso é necessário para que os sites indexadores e os motores de busca consigam atualizar as informações sobre sua conta, e posteriormente você não possa ser encontrado por sua última cópia. Além disso, não se esqueça de que a partir de agora você não pode deixar nenhum comentário e mensagens publicamente.
  3. Todos os comentários/tweets/mensagens antigas é aconselhável excluir. Existem mecanismos em redes sociais e mensageiros que permitem chegar até você mesmo após a exclusão: menções (um link para o seu perfil que parece um nome), uma marca de autoria, reposts/retweets. Vale a pena se livrar de todas essas pistas.
  4. Atualize os resultados da pesquisa pedindo-lhes para reindexar sua página modificada (Google) e espere um pouco para que outros sistemas que salvam informações de redes sociais também recebam uma nova versão da sua conta.

Depois disso, a conta pode ser excluída usando as ferramentas integradas da plataforma.

Solicitação de exclusão

Nem em todos os serviços, a exclusão é possível por meios regulares. Ou está escondido tão profundamente que é impossível encontrá-lo. Felizmente, existem sites auxiliares com informações sobre como se aposentar.

  • JustDeleteMe - um excelente diretório para plataformas, dá links e uma descrição das ações necessárias, mostra a complexidade da remoção. Os criadores também têm sites relacionados JustGetMyData e JustWhatsTheData para cumprir o parágrafo 1 acima.
  • AccountKiller - instruções de remoção para um grande número de sites, principalmente ocidentais.

Mesmo que você não tenha conseguido encontrar as instruções de remoção para o site que precisa, não desanime. Sempre há uma oportunidade de escrever para o suporte e solicitar a exclusão da sua conta de acordo com o GDPR.

Excluindo resultados de pesquisa em motores de busca

O Google oferece uma série de diferentes formulários através dos quais você pode solicitar a remoção e atualização de conteúdo nos resultados de pesquisa.

  • Removendo conteúdo desatualizado da pesquisa - bem adequado para páginas que precisam ser reindexadas (veja o ponto 4 acima sobre a exclusão de contas).
  • Formulário de Solicitação de Exclusão de Dados Pessoais do Google
  • Novo formulário de solicitação para exclusão de dados pessoais

Removendo informações sobre vazamentos

Instruções breves para excluir seus dados de serviços populares - Have I Been Pwned, Leakcheck, IntelX, etc.

Para fazer solicitações para serviços que estão interessados em armazenar seus dados, você precisa preparar uma lista completa de todas as informações que deseja limpar. Deve incluir:

  • apelidos
  • telefones
  • e-mails
  • endereços
  • placas de carro
  • identificadores governamentais (número do passaporte, número do seguro social, número de identificação do contribuinte, nome completo, etc.)
  • quaisquer dados de registros de banco de dados vazados.

Excluindo resultados de pesquisa em motores de busca

O TeleSINT oferece uma oportunidade para se esconder da busca, enviar alertas se tentarem te encontrar através do bot, e também ver a lista de quem estava procurando por você. Infelizmente, isso **requer uma assinatura paga**.

Bot do Telegram Quick OSINT

Na Política de Processamento de Dados Pessoais do serviço é dito:

O período de processamento de dados pessoais é ilimitado. O Usuário pode retirar seu consentimento para o processamento de dados pessoais a qualquer momento enviando uma notificação para o Operador @Quick_OSINT_support marcada como "Retirada do consentimento para o processamento de dados pessoais".

Todas as solicitações de exclusão são processadas manualmente, é necessário preencher um formulário para bloquear dados de fontes publicamente disponíveis com dados completos do passaporte, o endereço do bot, bem como identificadores para bloqueio, e então enviar sua foto ou imagem digitalizada para o Operador. Os dados serão excluídos dentro de 7 dias.

Bot do Telegram Insight

Para esconder dados sobre você no bot Insight é necessário:

  • ter pelo menos ₽1000 depósitos durante todo o tempo da conta
  • contatar @tginsighter com um pedido para excluir seus dados

Bot do Telegram GTA

align="justify"Para esconder dados sobre você no bot GTA (Global Tech Aggregator) é necessário:

  • iniciar o bot
  • "Мой аккаунт" - "Удалить информацию" - "Удалить запись"
  • inserir individualmente os identificadores de interesse
  • reabastecer a carteira em TON (para exclusão por um ano - 1 TON para 1 ID, permanentemente - 5 TON para 1 ID)

Bot do Telegram Zernenda

Para esconder dados sobre você no bot Zernenda é necessário:

  • iniciar o bot, verificar sua conta
  • verificar quais dados o bot sabe sobre você e de quais vazamentos (para fazer isso, pague várias solicitações para o seu número de telefone, nome completo, etc.; anote para si mesmo quais dados o bot encontrou)
  • "Мой аккаунт" - "Удалить данные о себе"
  • enviar seu número de telefone para confirmar sua conta
  • enviar um pedido detalhado indicando quais dados você possui

Formulário de solicitação de amostra:

Nome completo: Пушкин Владимир Владимирович
Telefone: +79999999999
Telegram: tg1569113435
VKontakte: vk10419593
Passaporte: 1234 567890
Eu confirmo que os dados acima são meus e peço que sejam removidos da pesquisa do bot.

Bot do Telegram Angel (Архангел)

Para esconder dados sobre você no bot Angel (Архангел) é necessário:

  • iniciar o bot
  • "Профиль" - "Приобрести запросы"
  • comprar solicitações (a opção "800 запросов - 5760 руб." é necessária)
  • iniciar o bot: https://t.me/AngelSystemSupportBot
  • escrever "Пополнил на 800 запросов. Прошу удалить мои данные из вашей системы (você deve escrever que tipo de dados - telefone, e-mails, redes sociais, documentos, endereços, etc.)"

Bot do Telegram Cerberus

Para esconder dados sobre você no bot Cerberus é necessário:

  • iniciar o bot
  • "Аккаунт" - "Защитить данные" - "Создать"
  • inserir os dados que lhe são solicitados um por um

GetContact

No GetContact, é possível desativar arbitrariamente a exibição de nomes que são conhecidos pelo seu número, ou desativar a pesquisa completamente.

Para fazer isso, siga este link e faça login via mensageiro, depois desative.
